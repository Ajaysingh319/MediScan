# 🩺 MediScan — AI-Powered Health Report Analyzer

**MediScan** is a smart healthcare web application that helps users understand their medical reports and get personalized preventive health advice.  
Users can **upload an image of their medical report** or **enter their report details as text**, and the system uses **OCR (Optical Character Recognition)** and **AI** to analyze it.  
It then provides **health insights, protection tips, and diet suggestions** to promote early awareness and better lifestyle decisions.

---

## 🚀 Features

- 📤 Upload medical report images (JPG, PNG)
- ✍️ Enter report values manually through text input
- 🧠 AI + OCR based analysis of health parameters
- ⚕️ Preventive advice and diet/lifestyle recommendations
- 📊 Simple and user-friendly interface
